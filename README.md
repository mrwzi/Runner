# Runner

https://github.com/mrwzi/Runner/releases/tag/v1.0.0

Runner is a Windows desktop app for managing multiple apps and scripts in one place.

## Features

* Add apps or scripts
* Start, stop, and restart processes
* Monitor status, PID, CPU, RAM, and uptime
* View and open log files
* Simple dark desktop interface

## Download

Go to the **Releases** section and download:

`RunnerSetup.exe`

## Usage

1. Install Runner.
2. Open the app.
3. Click **Add App**.
4. Select your app/script folder and runner file.
5. Use **Start**, **Stop**, or **Restart** to manage it.

## Notes

Runner is built for Windows.
